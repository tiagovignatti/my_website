# Images for /andre/ (The Machine of Nature)

Drop these files here and they appear on the page automatically (no code change):

- `cover.jpg` — the book cover, shown in the hero (portrait works best).
  Optional: if absent, the page falls back to the shared cover URL from
  `data/library.yaml` (the Goodreads/Amazon image). Add a local file only if
  you want higher quality or no external dependency.
- `andre.jpg` — a photo of André, shown in "About the author" (square works best).
  If absent, the photo slot is simply omitted.
